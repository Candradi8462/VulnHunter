## 🧰 Troubleshooting

### The app does not open

- Try running it again as a normal user
- Right-click and choose Run as administrator
- Check if your antivirus blocked the file
- Re-download the release if the file looks damaged

### The scan is slow

- Use a smaller target first
- Check your internet connection
- Close other network-heavy apps
- Try a quick scan mode

### No results show up

- Make sure the target URL is correct
- Check that the site is online
- Try another page on the same site
- Run the tool again after a short wait

### Windows flags the file

- Open the releases page again
- Make sure you downloaded the latest release
- Extract the file before opening it
- Allow the app if you trust the source and have permission to use it
